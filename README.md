# SafeAI-1

SafeAI-1 is a small homework & coding helper AI built from scratch in Python.  
It can answer simple math, coding, and science questions like a smart 12th grader.

## Features
- Neural network written from scratch (no pre-trained AI)
- Small dataset for homework/coding/science
- Command-line interface for asking questions
- Fully expandable dataset & neural network

## How to Run
1. Clone the repository:
git clone https://github.com/YOURUSERNAME/SafeAI-1.git

2. Navigate into the folder:


cd SafeAI-1

3. Install dependencies:


pip install -r requirements.txt

4. Run SafeAI-1:


python safeai.py

5. Ask questions! Type `exit` to quit.
