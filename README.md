# SafeAI-1

SafeAI-1 is a **hybrid AI agent** capable of:

- Symbolic reasoning
- Concept memory (SQLite)
- Neural-assisted intuition
- Autonomous planning
- Rule improvement and self-testing

## Features

- @learn: Add new concepts
- @improve: Add new rules
- Persistent memory
- Self-testing mode
- Logs thoughts to logs.txt

## Installation

```bash
git clone <repo_url>
cd safeai
python safeai.py
